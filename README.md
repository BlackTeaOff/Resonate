# Resonate Archive

一款以剧情向游戏为主题的视觉档案馆 Demo，主打暗黑赛博与玻璃拟态（Glassmorphism）风格。首页展示电影感宽屏卡片、毛玻璃信息层与悬浮导航栏。

## 功能亮点
- Material 3 暗黑主题 + 霓虹点缀
- 电影感宽屏卡片列表（覆盖真实 Steam 封面）
- 卡片底部毛玻璃信息层（标题/厂商）
- 悬浮液态玻璃底部导航栏（可交互）
- 顶部毛玻璃 Header（适配刘海屏）

## 运行方式
```bash
flutter pub get
flutter run
```

## 目录结构
目前主要逻辑在 `lib/main.dart`，后续可按模块拆分（如数据模型、组件、主题等）。
